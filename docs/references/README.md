# References

Lägg befintligt material här som agenterna ska utgå från i stället för att hitta på från noll.

## Vad som hör hemma här

**Tone & röst** (läses främst av `copywriter` och `brand-strategist`)
- Brand-bok, manifest, värderingar
- Tidigare copy som ni är nöjda med (ads, web, captions, taglines)
- Tone of voice-dokument
- Förbjudna ord / favoritord

**Visuellt** (läses främst av `art-director`)
- Logo-spec, färgpalett, typografi
- Moodboard-beskrivningar
- Fotostil-guider

Visuella **filer** (bilder) hör hemma i `assets/images/reference/` enligt CLAUDE.md — peka dit från en `.md` här om du vill ge agenterna kontext.

## Namnkonvention

Använd beskrivande filnamn:
- `tone-of-voice.md`
- `brand-book.md`
- `previous-copy-ads.md`
- `color-palette.md`

## När referenser saknas

Om en agent inte hittar relevanta referenser för sin uppgift ska den **flagga till `project-lead`** och fråga användaren — inte hitta på ton eller visuell riktning på egen hand.
